Hello World

A program to greet the planet.

This page is an attempt to greet everyone one Earth using various human languages. We're interested to see how long our list can become.

We'll start with languages that the Epicodus staff could offer without using Google.

  • English: Hello, World!
  • Spanish: Hola, Mundo!
  • Japanese: Konnichiwa, sekai!
  • French: Bonjour, monde!
  • Kinyarwanda: Mwire, isi!
  • German: Guten Tag, Welt!
  • Pig Latin: ello-hay, orld-way!
